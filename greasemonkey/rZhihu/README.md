# rZhihu - A Greasemonkey/Tampermonkey Script

Adding Google-like keyboard shortcuts for Zhihu homepage.

## features

|  **快捷键**  |          说明           |
|:-----------:|:-----------------------:|
|     J/K     |    上一个/下一个日报。     |
|   O/Enter   |      展开/收起日报。      |
|      U      |      不感兴趣/撤销。      |
|      V      |  在新标签页中打开当前回答。 |
